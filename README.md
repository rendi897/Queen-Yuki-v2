# RenzGG

Base on WhatsApp


## Run Locally

Clone project

```bash
  git clone https://github.com/NCTea/Felice-MD
```

Go to the project directory

```bash
  cd Queen-Yuki-v2
```

Install dependencies

```bash
  npm install
```

Install dependencies **recommended**
```bash
 npm install --force
```

Start the server

```bash
  npm start
```
